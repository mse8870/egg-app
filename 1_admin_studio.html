전체 패키지 다운로드

library.json + 모든 이미지 파일을 ZIP으로 묶어서 다운로드합니다.
GitHub Pages에 올리면 바로 서비스 가능합니다.

GitHub Pages 배포 가이드
1. GitHub 저장소 생성 (예: egg-library)
2. 전체 패키지 ZIP 압축 해제 후 저장소에 업로드
3. Settings → Pages → main 브랜치 배포
4. 사용자 앱에서 library.json URL 설정
5. 새 캐릭터 추가 시 JSON + 이미지만 추가하면 자동 반영
